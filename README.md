# Standalone_2AxisRCA
เครื่องมือ 2-Axis Swiss Cheese RCA with 4-Tier Contributing Factor
ใช้เป็นเครื่องมือช่วยขับเคลื่อนกระบวนการ RCA แบบ 2-Axis Swiss Cheese model 
ประกอบด้วย 4 Tier 10 กลุ่มปัจจัย 35 ปัจจัย ที่เป็นสาเหตุของการเกิดอุบัติการณ์ 
ตามนิยามของสถาบันรับรองคุณภาพสถานพยาบาล (สรพ.) ปีงบประมาณ 2569

ขั้นตอนประกอบด้วย
1. ระบุข้อมูลทั่วไปของอุบัติการณ์ที่ต้องการวิเคราะห์
2. ระบุ Flow of Step (แกน X)
3. ระบุ Unsafe Act / Potential Change ในแต่ละ Flow of Step (แกน X)
4. ระบุปัจจัย ที่สัมพันธ์กับแต่ละ Unsafe Act / Potential Change 
========== ข้อมูลที่จำเป็นต้องส่งเข้าสู่ระบบ NRLS ของ สรพ. สิ้นสุดที่นี่ ===========
5. ระบุ "ด่านกั้นที่ล้มเหลว โดยเลือกได้จากรายการที่สัมพันธ์กับปัจจัย (35 ปัจจัย 85 ด่านกั้น)
6. ระบุ แนวทางการแก้ไข (ถ้ามี)

พัฒนาโดย ภก.พิชญ์พงศ์  คงนาวัง 
กลุ่มงานพัฒนาคุณภาพบริการและมาตรฐาน
โรงพยาบาลจิตเวชเลยราชนครินทร์
